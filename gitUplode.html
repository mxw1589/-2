先进入仓库，Clone or dowload->点击本地项目右键 选择 Git Bash Here -> git clone https://github.com/CKTim/BlueTooth.git（https://github.com/CKTim/BlueTooth.git为上面复制的仓库链接） 然后把项目放到新出现的文件夹下 cd 新文件夹名 git add . （注：别忘记后面的.，此操作是把Test文件夹下面的文件都添加进来） git commit -m "提交信息" （注：“提交信息”里面换成你需要，如“first commit”） git push -u origin master （注：此操作目的是把本地仓库push到github上面，此步骤需要你输入帐号和密码）
